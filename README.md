# Aspenini.Fun

A game hub for my HTML web games!

![funky-icon](https://github.com/user-attachments/assets/10ef89c2-7453-4a29-b1ab-8df1327cd378)

## Overview

Aspenini.Fun is a curated game hub that provides easy access to a collection of HTML games. Designed for quick access and ease of use, you can play games directly online or download the desktop version for offline fun.

## Getting Started

1. **Access Options**:
   - **Web Version**: Instantly play online at [https://aspenini.github.io/Aspenini-Fun/](https://aspenini.github.io/Aspenini-Fun/).
   - **Desktop Version**: Download the Electron-packaged executable for an enhanced hub experience on your computer.

2. **Installation** (for Desktop):
   - Download the `.exe` file and follow the prompts to install.

3. **System Requirements**:
   - Web: Works on all modern browsers.
   - Desktop: Requires Windows 7 or higher (for the EXE version).

## Features

- **Preinstalled Games**: Enjoy a selection of games, no setup required.
- **Multiple Platforms**: Access games through the web or as a desktop application.
- **User-Friendly Layout**: Simple and intuitive interface with easy navigation between games.

## Development

Built using HTML and JavaScript, with Electron for the desktop experience.

### Project Structure

- **assets/** - Contains images, icons, and other resources.
- **src/** - Main source code for the hub and games.
- **package.json** - Node.js dependencies and configuration.
